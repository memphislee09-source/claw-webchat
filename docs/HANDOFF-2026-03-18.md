# Handoff — 2026-03-18

## 当前结论
`openclaw-webchat` 当前应以 `main` 为唯一继续开发基线；2026-03-18 这一轮代码审查修复、体验修正和历史搜索第一版已经并入主线，不再需要回到单独修复分支继续维护。

当前已确认的主线最新状态：
- 当前版本已提升到 `0.1.3`
- 服务默认地址仍为 `http://127.0.0.1:3770`
- 当前开发基点为 `main`
- 用户级 LaunchAgent `ai.openclaw.webchat` 继续作为常驻运行方式
- 2026-03-18 的安全/稳定性修复已纳入基线：
  - 媒体签名 secret 改为 data 目录持久化随机 secret
  - 本地媒体白名单收紧到上传目录与 `~/.openclaw`
  - 文件类媒体链接只允许 `http/https` 或受控本地签名地址
  - assistant 超时占位不再污染正式历史
  - 音频转写改为异步子进程执行
  - 自测脚本统一请求超时控制
- 2026-03-18 的体验修正也已纳入基线：
  - 聊天页自动跟随滚动补统一贴底状态管理
  - 历史头像 token 保留受限兼容迁移
  - 左侧 agent 列表改为局部刷新，身份信息由设置保存时主动推送，轮询只刷新状态/摘要/时间
- 2026-03-18 的历史搜索能力已纳入基线：
  - 当前 agent 主时间线关键词搜索与命中跳转
  - 顶部常驻搜索框、外部点击收起与稳定跳转
  - 搜索结果点击后保持展开，支持当前 agent 维度最近搜索回选
  - 命中消息关键词高亮，最近搜索下拉改为更简洁的列表样式
